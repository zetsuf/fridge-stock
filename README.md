# 冷蔵庫の在庫 (fridge-stock)

冷蔵庫の中身を管理するモバイル向けアプリ。HTML/CSS/JS で作成し、Capacitor で Android アプリ(APK)化する想定。

## 機能
- メイン画面で **名称・残量・タグ・最終使用日** を一目で確認
- 残量は **5段階スライダー**（未使用／ほぼ新品／半分／後わずか／なし）で管理
- タグは **6種固定**（野菜・肉・調味料・酒・氷・その他）から複数選択
- 右下のフローティング＋ボタン（**スクロール中は隠れ、停止で再出現**）から追加
- アイテムをタップで編集 / 削除
- 「使った」ボタンで編集シートを開き、残量スライダーの調整・削除・保存ができる（保存で最終使用日を更新）
- **複数の在庫ページ**に対応（冷蔵庫／冷凍庫／パントリーなどを切り替え・追加・削除）
- タグでフィルタ、名称・タグで検索
- 7日以上使っていないアイテムは最終使用日を赤字で警告
- データは端末内に保存（localStorage）。PWA対応でホーム画面に追加・オフライン起動可

## 開発プレビュー
```bash
node server.js
# http://localhost:5188
```

## ファイル構成
- `index.html` — 画面構造
- `styles.css` — スタイル（モバイル最適化・セーフエリア対応）
- `app.js` — データ管理と UI ロジック
- `server.js` — 開発用の静的サーバ（依存なし）

## Android アプリ化（次のステップ）
Capacitor を使って APK を生成する。別途 JDK 17+ と Android SDK が必要。
